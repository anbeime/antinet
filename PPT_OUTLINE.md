## 幻灯片8：算力单元选择理由（✅ 已实现）

### 为什么选择Hexagon NPU？

**1️⃣ 低功耗优势**
- NPU能效比CPU高10-20倍
- 适合移动端和AIPC场景
- 延长设备续航时间

**2️⃣ 高性能推理**
- 专用AI加速器，并行计算能力强
- INT8量化支持，模型体积减小75%
- BURST模式优化，降低延迟

**3️⃣ 数据安全与隐私保护**
- ✅ 端侧处理，数据不出域
- ✅ 零云端依赖
- ✅ 符合隐私法规要求

**4️⃣ 异构计算优化**
- CPU：API服务、数据库操作
- NPU：大模型推理（7B参数）
- GPU：可选加速（当前未使用）
- 内存：模型缓存 + 知识库

### 当前异构计算架构

```
查询请求
   ↓
┌─────────────────────────────┐
│  CPU（FastAPI路由层）        │
│  - 请求接收与验证            │
│  - 查询复杂度评估            │
│  - 模型选择路由              │
└──────────────┬──────────────┘
               ↓
        ┌──────┴──────┐
        │  路由决策    │
        └──────┬──────┘
               ↓
    ┌──────────┴──────────┐
    ↓                     ↓
┌───┴────┐          ┌────┴────┐
│ 3B模型 │          │ 7B模型  │
│ CPU    │          │ NPU     │
└───┬────┘          └────┬────┘
    ↓                    ↓
┌───┴────┐          ┌────┴────┐
│ 简单查询│          │复杂分析 │
│ <30分  │          │ >60分   │
└────────┘          └─────────┘
    ↓                    ↓
┌───┴────────────────────┴────┐
│  CPU（响应处理层）           │
│  - 四色卡片格式化            │
│  - 结果返回                  │
```
