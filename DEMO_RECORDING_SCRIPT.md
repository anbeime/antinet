# Antinet 知识管理系统 - DEMO 录制脚本

## 🎬 录制准备

### 前置条件检查
- [ ] 后端服务运行：http://localhost:8000
- [ ] 前端服务运行：http://localhost:3000
- [ ] 浏览器已打开并访问 http://localhost:3000
- [ ] 录屏软件已准备（推荐：OBS Studio / Windows Game Bar）

### 录制设置建议
- **分辨率：** 1920x1080 或 1280x720
- **帧率：** 30 FPS
- **时长：** 5-8 分钟
- **格式：** MP4

---

## 📝 DEMO 录制脚本（详细版）

### 第一部分：系统介绍与首页展示（1-2分钟）

#### 场景 1：打开首页并介绍
**操作步骤：**
1. 打开浏览器，访问 http://localhost:3000
2. **停留 3 秒** - 展示完整首页界面
3. **画外音（可选）：** "这是 Antinet 智能知识管家，基于卢曼卡片盒方法的 AI 驱动企业知识管理系统"

#### 场景 2：展示知识概览统计
**操作步骤：**
1. **停留在顶部统计区域 2 秒**
2. 用鼠标指针指向统计数字：
   - 核心概念：15
   - 关联链接：8
   - 参考来源：8
   - 索引关键词：19
3. **画外音（可选）：** "系统当前管理着 50 张知识卡片，按四色分类法组织"

#### 场景 3：展示最近知识活动
**操作步骤：**
1. **缓慢向下滚动** 到"最近知识活动"区域
2. **停留 3 秒** - 展示活动列表
3. 用鼠标悬停在几个卡片上（不点击）

#### 场景 4：展示知识分布图表
**操作步骤：**
1. **继续向下滚动** 到知识分布图表
2. **停留 3 秒** - 展示图表
3. **向下滚动** 展示特性亮点和应用场景
4. **滚动回顶部**

---

### 第二部分：创建新卡片（2-3分钟）

#### 场景 5：打开创建卡片弹窗
**操作步骤：**
1. **点击** 右上角"新建卡片"按钮（蓝色按钮）
2. **等待 1 秒** - 弹窗出现
3. **停留 2 秒** - 展示完整的创建表单

#### 场景 6：选择卡片类型
**操作步骤：**
1. **点击** "核心概念"选项（蓝色图标）
2. **停留 1 秒** - 显示选中状态
3. **画外音（可选）：** "选择核心概念类型，用于记录重要的想法和理论"

#### 场景 7：填写卡片标题
**操作步骤：**
1. **点击** "卡片标题"输入框
2. **逐字输入**（不要粘贴）："AI驱动的知识管理"
3. **停留 1 秒**

#### 场景 8：填写卡片内容
**操作步骤：**
1. **点击** "卡片内容"输入框
2. **逐字输入**（不要粘贴）：
   ```
   使用NPU加速的端侧智能知识管理系统，支持四色卡片分类和8-Agent协同工作。
   数据完全本地处理，推理延迟小于500ms。
   ```
3. **停留 1 秒**

#### 场景 9：生成卡片地址
**操作步骤：**
1. **点击** "自动生成地址"按钮
2. **等待 1 秒** - 地址自动填充（如：A17）
3. **停留 1 秒** - 展示生成的地址

#### 场景 10：创建卡片
**操作步骤：**
1. **点击** "创建卡片"按钮（底部蓝色按钮）
2. **等待 2 秒** - 等待成功提示出现
3. **观察** 页面顶部的成功提示："卡片创建成功"
4. **观察** 统计数字变化（核心概念：15 → 16）
5. **观察** 最近活动列表顶部出现新记录

---

### 第三部分：搜索功能演示（1分钟）

## 🎯 录制技巧

### 操作节奏
- **慢速操作**：每个点击、输入都要慢，确保观众看清
- **停留时间**：每个页面/功能至少停留 2-3 秒
- **平滑滚动**：滚动页面要缓慢平滑
- **鼠标指引**：用鼠标指针指向重要内容

### 画面质量
- **全屏录制**：确保浏览器全屏或最大化
- **隐藏干扰**：关闭其他标签页、通知
- **清晰度**：确保文字清晰可读
- **光标可见**：确保鼠标光标在录制中可见

### 音频（可选）
- **背景音乐**：轻柔的科技感音乐
- **画外音**：清晰的功能介绍
- **无音频**：纯演示也可以

## 🎬 开始录制！

### 录制流程
1. **准备阶段**（录制前）
   - 打开浏览器，访问 http://localhost:3000
   - 确保页面完全加载
   - 关闭所有通知和弹窗
   - 鼠标移到屏幕外或左上角

2. **开始录制**
   - 启动录屏软件
   - 等待 2 秒（倒计时）
   - 开始按脚本操作

3. **录制中**
   - 严格按照脚本操作
   - 每个操作之间停留 1-2 秒
   - 保持操作流畅自然
   - 如有失误，继续录制（后期可剪辑）

4. **结束录制**
   - 完成最后一个场景
   - 停留 3 秒
   - 停止录制

5. **后期处理**（可选）
   - 剪辑掉失误部分
   - 添加片头片尾
   - 添加背景音乐
   - 添加字幕说明
   - 导出最终视频
