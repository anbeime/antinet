deploy_antinet.bat - 一键完整部署
  ├─ 自动检查环境（Python、Node.js、pnpm）
  ├─ 自动配置 NPU 环境
  ├─ 自动安装后端依赖

  ├─ 启动后端服务（新窗口）

stop_all.bat - 停止服务
  ├─ 停止后端服务（端口8000）
  ├─ 停止前端服务（端口3000）
  ├─ 清理相关进程
  └─ 验证端口释放

health_check.bat - 健康检查
  ├─ 检查环境（Python、Node.js、pnpm）

  ├─ 检查 API 健康度
