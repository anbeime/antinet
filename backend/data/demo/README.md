# Antinet 演示数据集

这个目录包含用于演示的样例数据，展示系统的数据分析能力。

## 数据集说明

### 1. sales_data.csv - 销售数据
模拟企业月度销售数据，包含：
- 时间维度
- 产品类别
- 销售额
- 客户数量
- 地区分布

### 2. customer_feedback.csv - 客户反馈
模拟客户反馈数据，用于情感分析和问题识别

### 3. market_trends.csv - 市场趋势
模拟行业市场趋势数据

## 使用方法

在数据分析面板中：
1. 点击"上传数据"
2. 选择样例CSV文件
3. 使用自然语言查询，例如：
   - "分析上个月的销售数据趋势"
   - "客户反馈中的主要问题是什么"
   - "哪个地区的销售增长最快"

## 演示视频建议

推荐展示的查询：
1. **销售分析**: "对比Q1和Q2的销售业绩"
2. **客户洞察**: "总结客户最关心的前3个问题"
3. **趋势预测**: "根据历史数据预测下月销售额"
