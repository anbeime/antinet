# 🎬 Antinet 录屏演示操作指南

## 📋 准备工作

### 1. 确认服务状态
- ✅ 后端：http://localhost:8000 （NPU 模型已加载）
- ✅ 前端：http://localhost:3000
- ✅ 测试数据：`data/test_sales_data.csv` 已准备

### 2. 录屏工具
**推荐：** Windows 游戏栏（Win+G）
- 按 `Win + G` 打开游戏栏
- 点击录制按钮开始录屏
- 按 `Win + Alt + R` 停止录屏

**备选：** OBS Studio

### 3. 浏览器准备
- 打开 Chrome/Edge
- 准备两个标签页：
  - 标签页 1：http://localhost:3000 （前端）
  - 标签页 2：http://localhost:8000/docs （API 文档）

---

## 🎯 演示脚本（10分钟）

### 开场白（30秒）
```
大家好，我是 [你的名字]。
今天为大家演示 Antinet 智能知识管家系统。

这是一个基于 NPU 端侧 AI 的企业知识管理平台，
采用卢曼卡片盒方法和四色卡片系统，
实现数据不出域的智能分析。

让我们开始演示。
```

---

### 第一部分：系统概览（2分钟）

#### 1.1 打开前端页面
1. 访问 http://localhost:3000
2. 展示主页界面
3. 介绍主要功能模块

**解说词：**
```
这是 Antinet 的主界面。
左侧是导航菜单，包含：
- 首页：系统概览
- NPU 仪表板：性能监控
- 数据管理：导入导出
- 智能分析：AI 驱动的数据分析
- 知识库：卡片管理
```

#### 1.2 查看 NPU 状态
1. 点击 "NPU Dashboard" 或 "性能监控"
2. 展示 NPU 信息

**解说词：**
```
这里显示了 NPU 的状态信息：
- 设备：Qualcomm Hexagon NPU
- 模型：Qwen2.0-7B-SSD
- 参数量：7B
- 状态：已加载

这是端侧 AI 的核心，所有推理都在本地完成。
```

---

### 第二部分：数据导入（2分钟）

#### 2.1 导入 CSV 数据
1. 点击 "数据管理" 或 "导入数据"
2. 点击 "上传文件" 按钮
3. 选择 `data/test_sales_data.csv`
4. 等待上传完成
5. 查看数据预览

**解说词：**
```
现在我们导入一份销售数据。
这是一个包含产品销售信息的 CSV 文件。

[上传文件]

系统正在解析数据...
可以看到数据预览，包含产品名称、销售数量、销售额等字段。

数据导入成功！
```

---

### 第三部分：智能分析（NPU 推理）（3分钟）

#### 3.1 发起智能分析
1. 选择已导入的数据
2. 点击 "智能分析" 或 "AI 分析"
3. 在输入框输入：
   ```
   分析这份销售数据，识别销售趋势、潜在风险，并提出改进建议
   ```
4. 点击 "开始分析"
5. 观察 NPU 推理过程

**解说词：**
```
现在我们使用 NPU 进行智能分析。

我输入的分析需求是：
"分析这份销售数据，识别销售趋势、潜在风险，并提出改进建议"

点击开始分析...

[等待 NPU 推理]

系统正在使用 NPU 进行推理，
这个过程完全在本地完成，数据不会上传到云端。
```

#### 3.2 查看分析结果
1. 等待分析完成（约 5-10 秒）
2. 查看生成的四色卡片

**解说词：**
```
分析完成！系统生成了四色卡片：

[展示蓝色卡片 - 事实]
蓝色卡片记录了数据事实：
- 总销售额
- 销售数量
- 时间范围

[展示绿色卡片 - 解释]
绿色卡片提供了趋势解释：
- 销售增长趋势
- 产品类别分析
- 区域分布

[展示红色卡片 - 风险]
红色卡片识别了潜在风险：
- 库存风险
- 市场风险

[展示黄色卡片 - 行动]
黄色卡片提出了行动建议：
- 优化策略
- 改进方案
```

---

```
输入关键词 "销售"...

还可以按类型、时间、标签进行过滤。
```

## 🎥 录屏技巧

### 1. 画面设置
- 分辨率：1920x1080
- 帧率：30 FPS
- 窗口模式：全屏或最大化浏览器

### 2. 操作技巧
- 鼠标移动要流畅
- 点击要明确
- 等待加载完成再继续
- 突出重点功能

### 3. 解说技巧
- 语速适中
- 发音清晰
- 强调关键点：NPU、端侧、数据不出域

### 4. 时间控制
- 总时长：10-15 分钟
- 每个部分控制在 2-3 分钟
- 留出缓冲时间

---

## 🐛 常见问题处理

### 问题 1：页面加载慢
**解决：** 刷新页面，清除缓存

### 问题 2：文件上传失败
**解决：** 检查文件路径，确认文件存在

### 问题 3：NPU 推理超时
**解决：** 减少输入长度，简化分析需求

### 问题 4：卡片不显示
**解决：** 检查后端日志，确认数据已保存
